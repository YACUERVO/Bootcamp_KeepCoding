# Búsqueda Lineal en Python

La búsqueda lineal es un algoritmo fundamental para encontrar un elemento específico dentro de una lista. Comprender este concepto es crucial antes de explorar algoritmos de búsqueda más avanzados, ya que sienta las bases para optimizar procesos en programación.

## ¿Qué es la Búsqueda Lineal?

La búsqueda lineal (también conocida como búsqueda secuencial) examina cada elemento de una lista en orden, comparándolo con el valor buscado. El proceso continúa hasta que se localiza el elemento o se recorre toda la lista.  Es un método simple pero a menudo ineficiente para grandes conjuntos de datos.

## Funcionamiento y Consideraciones

La eficiencia de la búsqueda lineal depende del tamaño de la lista. En el mejor de los casos (el elemento buscado es el primero), se requiere un solo paso. En el peor caso (el elemento no está o es el último), se debe examinar toda la lista.  El caso promedio implica revisar la mitad de los elementos.

**Conceptos clave:**

* **Simplicidad:** Fácil implementación a través de bucles.
* **Complejidad:**  Ineficiente para listas grandes.
* **Orden:** Funciona correctamente con listas ordenadas o desordenadas.

La búsqueda lineal es un pilar en el desarrollo de algoritmos y una herramienta valiosa para entender la lógica subyacente a las búsquedas en programación.
