# Secuencias

Esta sección introduce las secuencias, una estructura de datos fundamental en programación. Comprenderlas es crucial para construir algoritmos eficientes y manejar datos de manera organizada. Las secuencias permiten acceder a elementos específicos, iterar sobre ellos y son la base para estructuras de datos más avanzadas.

## ¿Qué son las Secuencias?

Una secuencia es una colección ordenada de elementos.  Su principal propósito es facilitar el acceso y manipulación de datos, transformando tareas complejas en procesos más sencillos.  Imagínese una fila en un supermercado: permite atender a los clientes de forma ordenada, asegurando que nadie sea olvidado. Las secuencias abstractan la ubicación física de los elementos, permitiendo acceder a ellos mediante su posición.

## Características Clave

* **Longitud:** Todas las secuencias tienen un número definido de elementos.
* **Acceso por Posición:** Se puede acceder a cualquier elemento dentro de la secuencia utilizando su posición (índice).
* **Orden:** El orden de los elementos es significativo y se mantiene dentro de la secuencia.
* **Iteración:**  Es posible recorrer todos los elementos de una secuencia, uno por uno.
