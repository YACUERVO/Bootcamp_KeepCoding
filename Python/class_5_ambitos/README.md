# Ambito en Python

Este módulo explora el concepto de ámbito en Python, fundamental para entender cómo las variables y funciones son accesibles en diferentes partes del código. Comprender los ámbitos es crucial para evitar errores y escribir programas bien estructurados, especialmente al trabajar con funciones y módulos.

## Conceptos Fundamentales

El ámbito define la región del código donde una variable o función es accesible. Python distingue principalmente dos tipos de ámbitos: global y local. El ámbito global se extiende a todo el programa, mientras que el ámbito local está restringido al contexto donde se define (por ejemplo, dentro de una función). El conocimiento de estos ámbitos es esencial para gestionar el alcance y la visibilidad de las variables.

## Ámbito Global vs. Local

El ámbito global es accesible desde cualquier parte del código, mientras que el ámbito local se crea dentro de funciones y tiene una visibilidad limitada. Las variables locales anidan en ámbitos más amplios, permitiendo la reutilización y organización del código. La visibilidad de las variables depende de su ámbito, lo que permite manipular datos dentro de funciones sin afectar variables globales accidentales.

**Conceptos Clave:**

* Ámbito: Región de accesibilidad de variables y funciones.
* Ámbito Global: Accesible en todo el programa.
* Ámbito Local:  Limitado a la función/bloque donde se define.
* Visibilidad: Depende del ámbito de las variables y funciones.
