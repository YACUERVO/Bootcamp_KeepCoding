# Funciones y Abstracción de Función

Este módulo introduce el concepto fundamental de abstracción a través de las funciones en Python. Aprender a definir y utilizar funciones es esencial para escribir código modular, reutilizable y más fácil de entender.  La abstracción con funciones permite organizar el código en bloques lógicos, simplificando la complejidad y promoviendo la eficiencia.

## ¿Qué son las Funciones?

Las funciones son bloques reutilizables de código diseñados para realizar una tarea específica.  En lugar de repetir código, las funciones permiten definir una secuencia de instrucciones y llamarla desde múltiples lugares. Esto mejora la legibilidad, reduce la redundancia y facilita el mantenimiento del código.

## Definición de Funciones en Python

La sintaxis básica para definir una función es:

```python
def nombre_funcion(parametros):
    cuerpo_funcion
    return valor  # Opcional
* `def`: Palabra reservada que indica el inicio de la definición.
* `nombre_funcion`:  Nombre descriptivo para identificar la función.
* `parametros`: Valores opcionales que se pasan a la función.
* `cuerpo_funcion`:  Código que se ejecuta cuando se llama a la función.
* `return`:  Devuelve un valor (opcional).

**Conceptos Clave:**

* **Abstracción:** Ocultar la implementación interna de una función y exponer solo su interfaz.
* **Reutilización:**  Las funciones se pueden llamar varias veces con diferentes parámetros.
* **Modularidad:** Las funciones dividen un programa en partes más pequeñas y manejables.
* **Legibilidad:**  Las funciones mejoran la claridad del código al dividir tareas complejas.
* **Parámetros:** Valores que una función recibe para realizar su tarea.
