# Pereza Productiva

Este módulo introduce los conceptos de variables y abstracciones, pilares fundamentales en la programación.  Comprender cómo asignar nombres a datos y abstraer información es crucial para escribir código legible, reutilizable y mantenible.  La "pereza productiva" en programación implica encontrar formas de simplificar tareas y evitar la repetición, lo cual se logra a través del uso efectivo de estas técnicas.

## Variables

Una variable es un nombre que se asocia a un valor.  Permite referirse al valor sin tener que escribirlo directamente en el código cada vez que se necesita. Esto mejora la legibilidad y facilita modificaciones futuras.

* **Definición:** Asignar un nombre a un valor.
* **Valor:** La información que almacena la variable.
* **Reutilización:** Las variables permiten usar un valor varias veces sin repetición.
* **Tipos:** Variables pueden almacenar diferentes tipos de datos (números, texto, etc.).
* **Nombres:** Deben ser descriptivos y seguir reglas de nomenclatura (comenzar con letra, usar guiones bajos).

## Abstracción

La abstracción es la técnica de ocultar detalles complejos y presentar solo lo esencial.  Permite simplificar problemas al enfocarse en la información más relevante, ignorando detalles innecesarios.  Esto facilita el diseño de sistemas modulares y la reutilización de código.

* **Concepto:** Simplificación al enfocarse en lo esencial.
* **Beneficios:** Reduce la complejidad, mejora la reutilización y facilita el mantenimiento.
* **Ejemplos:**  "Coche" es una abstracción que engloba diferentes marcas y modelos.
* **Proceso:** Identificar la esencia de un objeto o concepto y crear una representación simplificada.

## Abstracción de Variables

La abstracción de variables implica usar un nombre (la variable) para representar un valor, en lugar de referirse directamente al valor en el código. Esto mejora la legibilidad y facilita modificaciones futuras del código, ya que se puede cambiar el valor de la variable sin modificar todas las partes del programa donde se usa.
