# Secuencias Implícitas

Este módulo introduce el concepto de secuencias en programación, un tema fundamental para la manipulación de datos. Comprender las secuencias es crucial para escribir código eficiente y legible, permitiendo procesar información organizada de manera sistemática.

## Conceptos Fundamentales

Una secuencia es una estructura de datos que contiene elementos ordenados, accesibles por su posición (índice).  Python proporciona herramientas para iterar fácilmente sobre estas secuencias, simplificando la manipulación de sus elementos. El uso correcto de bucles `for` es clave para acceder a cada elemento, evitando errores comunes relacionados con el rango de la secuencia.

## Secuencias Explícitas vs. Implícitas

Las secuencias se clasifican en explícitas (listas, cadenas) e implícitas. Las secuencias explícitas son estructuras de datos que se almacenan en variables y pueden ser modificadas.  Las secuencias implícitas, como las que se explorarán en el ejemplo de números primos, requieren un enfoque diferente para su generación y manipulación.

### Predicados y Divisiones

El ejemplo de números primos ilustra cómo crear funciones (predicados) que operan sobre secuencias implícitas. Se introduce el concepto de división entera (`//`) y resto de división (`%`) como herramientas esenciales para determinar la primalidad de un número, demostrando su aplicación práctica en algoritmos.

**Conceptos Clave:**

* Secuencia: Orden de datos accesible por índice.
* Iteración: Recorrer los elementos de una secuencia.
* Secuencia Explícita: Lista o cadena almacenada en variable.
* Secuencia Implícita: Secuencia generada dinámicamente (ej., números primos).
* División Entera (`//`):  Devuelve la parte entera del resultado.
* Resto de División (`%`): Devuelve el residuo de la división.
