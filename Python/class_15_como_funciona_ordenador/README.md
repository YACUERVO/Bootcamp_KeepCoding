# Cómo Funciona un Ordenador: Una Visión General

Este módulo explora la arquitectura básica de un ordenador y cómo los diferentes componentes colaboran para procesar información. Comprender estos conceptos es fundamental para optimizar el rendimiento de programas y entender las limitaciones del hardware.

## Componentes Clave y Velocidad

La velocidad de los componentes de un ordenador varía significativamente.  Los registros, la RAM y el almacenamiento masivo tienen diferentes escalas de velocidad. Entendiendo estas diferencias, podemos mejorar la eficiencia del sistema.

### Registros: El Procesamiento Instantáneo

Los registros son la unidad de procesamiento más rápida. Almacenan datos directamente accesibles para la Unidad Central de Procesamiento (CPU).  Es como tener un documento listo sobre tu escritorio: acceso inmediato.

### RAM: Almacenamiento Temporal de Alta Velocidad

La memoria RAM (Random Access Memory) sirve como almacenamiento temporal para los datos que la CPU está utilizando activamente. Es más lenta que los registros, pero permite acceder a muchos datos simultáneamente. Imagina un armario cercano en tu despacho: más rápido que el escritorio, pero requiriendo una pequeña búsqueda.

### Almacenamiento Masivo: El Almacenamiento a Largo Plazo

El almacenamiento masivo (SSD, HDD) se utiliza para almacenar datos de forma permanente. Es el componente más lento y requiere un proceso más complejo para acceder a la información.  Piensa en un armario en otro despacho, o incluso en otra ubicación: más lento y con una mayor latencia.

### La Importancia de la Cercanía de los Datos a la CPU

El rendimiento general depende en gran medida de la velocidad con la que la CPU puede acceder a los datos.  Cuanto más cerca estén los datos de la CPU, más rápido será el procesamiento.

**Conceptos Principales:**

* Registros: Almacenamiento de datos de acceso inmediato.
* RAM: Almacenamiento temporal de alta velocidad.
* Almacenamiento Masivo: Almacenamiento permanente, pero más lento.
* Velocidad de los Componentes: Impacto en el rendimiento general del sistema.
