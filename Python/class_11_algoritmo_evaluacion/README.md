# Algoritmos de Evaluación

La evaluación es el proceso por el cual Python (y otros lenguajes) transforman tu código en un valor interpretable. Este valor es el resultado final de la ejecución del programa. 

El objetivo principal es reducir todo a un único `valor`, representando el estado resultante del código.  Entender este proceso es crucial para depurar y comprender cómo funciona cualquier programa.

## Elementos Fundamentales

El proceso de evaluación implica la manipulación de diferentes *elementos* del programa, que se evalúan en un orden específico. Estos elementos son los bloques constructivos de cualquier programa y permiten la creación de estructuras complejas.

Los elementos principales, ordenados de más simple a más complejo son:

* **Átomos:** Los elementos más básicos, que no pueden dividirse.
* **Expresiones:** Combinaciones de átomos y funciones.
* **Funciones:** Bloques reutilizables con un valor propio. 

Comprender cómo se evalúan estos elementos es clave para entender la ejecución del código en Python.
